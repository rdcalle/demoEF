Demo para EvoluFarma desarrollado en MEAN.JS

Para utilizar esta demo se pueden realizar las siguientes opciones:

  • Entrar, para acceder al login de acceso y validarse
  • Registrarse, para registrarse y poder entrar a la aplicación

Sólo cuando esté logueado se podrá:

  • Pulsar en 'Datos de Usuario', para ver en detalle sus datos
  • Pulsar en 'Trabajar', para introducir datos
  • Pulsar en 'Salir', para abandonar la aplicación
